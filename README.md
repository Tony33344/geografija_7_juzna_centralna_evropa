# Geografija 7. razred — Interaktivni učni pripomoček

Spletna aplikacija za pripravo na spraševanje iz geografije za 7. razred osnovne šole (Slovenija).

## Vsebina

Aplikacija pokriva vso snov o **Južni, Jugovzhodni in Srednji Evropi** ter splošnih značilnostih Evrope:

- **Učbenik** — strukturirana snov z glosarjem in označevanjem napredka
- **Interaktivni zemljevid** — klikljivi zemljevid s filtri in igro "Najdi"
- **Kvizi** — 116+ vprašanj (izberi enega, izberi vse, res/ni res)
- **Kartice (flashcards)** — 88 kartic s sistemom Leitner za ponavljanje
- **Simulator ustnega spraševanja** — 64 vprašanj z model odgovori in samoocenjevanjem
- **Mini-igre** — višinski pasovi v Alpah, sestavi vulkan, razpredelnica onesnaževanja
- **Iskanje** — iskanje po celotni vsebini
- **Listek za na pamet** — natisni prijazen pregled pojmov in vprašanj
